# GenshinDictionary

原神キャラクター図鑑

# 機能
## **キャラクター図鑑**
- キャラクターの名前、元素、使用武器、使用素材の早見表を確認できます。 
- 名前、元素、武器、所属国名による絞り込みが可能です。
- 天賦育成素材には、秘境で獲得可能な曜日を併記しています。

## **\[WIP\]育成素材逆引き表**
- 育成素材ごとに、誰がその素材を使用するかを確認することができます。

## **\[WIP\]イベントスケジュール表**
- ゲーム内各種イベントやバージョン更新の開始終了日時やカウントダウンなどを確認することができます。
